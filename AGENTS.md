# Eberban AI Knowledge Base

This repository contains an AI-readable knowledge base for the Eberban logical constructed language.

## Knowledge Base Location

All AI context files are in [`.ai/eberban-expert/`](.ai/eberban-expert/):

- [`SKILL.md`](.ai/eberban-expert/SKILL.md) — main reference: grammar, morphology, vocabulary, binding, design decisions
- [`references/grammar-deep-dive.md`](.ai/eberban-expert/references/grammar-deep-dive.md) — detailed morphology, logic framework, binding mechanics, time system
- [`references/particle-families.md`](.ai/eberban-expert/references/particle-families.md) — complete particle family inventory with prefix patterns
- [`references/numbers-quotes-vocab.md`](.ai/eberban-expert/references/numbers-quotes-vocab.md) — number system, quotes, enumeration, colors, dictionary conventions
- [`references/worked-examples.md`](.ai/eberban-expert/references/worked-examples.md) — parsed sentence traces, common patterns, key lessons
- [`references/vocabulary-gaps.md`](.ai/eberban-expert/references/vocabulary-gaps.md) — domain coverage analysis, expansion priorities
- [`references/writing-and-tooling.md`](.ai/eberban-expert/references/writing-and-tooling.md) — writing systems, web tooling, parser, dev workflows
- [`references/open-design-problems.md`](.ai/eberban-expert/references/open-design-problems.md) — unsolved design problems

Start with `SKILL.md` for an overview, then load reference files as needed for detailed information.
